# 第二章 核心问题与解题思路

## 2.1 问题全景

后货币时代面临的核心问题可以归纳为一个根本矛盾：

**当生存不再是劳动的动力，社会如何激励贡献、记录贡献、并基于贡献进行治理？**

这个根本矛盾分解为四个子问题：

### 问题一：激励

没有货币回报，人们为什么还要付出？

答案：社会认同。人类天然渴望被认可、被尊重、拥有影响力。GMC 将贡献转化为可见的、可量化的社会认同（MeriToken），并赋予其实际的话语权。

### 问题二：度量

贡献如何被公正地记录和衡量？

答案：去中心化的共识记录。借助区块链和加密技术，建立不可伪造、不可篡改、不可否认的贡献记录网络。没有任何单一机构可以扭曲记录。

### 问题三：治理

话语权应该基于什么分配？

答案：基于贡献度的动态分配。话语权来自 MeriToken 在社群中的占比，而 MeriToken 会衰减——这意味着话语权必须通过持续贡献来维持，不存在永久的特权。

### 问题四：人机协同

当 Fay 承担了大部分工作，如何公平衡量人类和 Fay 的贡献？

答案：统一的度量体系。人类和 Fay 在同一个系统中被记录和衡量，Fay 的贡献归属于其人类原型或所属组织。

## 2.2 解题思路

GMC 的解题思路可以概括为：

```
贡献 → 记录 → 度量 → 话语权
```

1. **贡献发生**：人类或 Fay 做出了对社会有价值的行为
2. **贡献被记录**：通过共识机制，贡献被不可篡改地记录在链上
3. **贡献被度量**：贡献转化为 MeriToken，有衰减、有底部值
4. **话语权生效**：MeriToken 占比决定在社群治理中的影响力

## 2.3 六大设计原则

| 原则 | 含义 | 解决的问题 |
|------|------|-----------|
| 不可伪造 | 贡献记录必须经过共识验证 | 防止虚假贡献 |
| 不可篡改 | 历史记录永久保存，无法修改 | 保证记录公信力 |
| 不可否认 | 贡献者无法否认自己的行为 | 责任可追溯 |
| 去中心化 | 不依赖单一机构 | 防止权力垄断 |
| 动态衡量 | 贡献度随时间衰减 | 激励持续参与 |
| 全民参与 | 面向所有人类和 Fay | 公平性 |

## 2.4 与货币体系的本质区别

| 维度 | 货币 | MeriToken |
|------|------|-----------|
| 本质 | 交换媒介 | 贡献度量 |
| 驱动力 | 生存需求 | 社会认同与自我实现 |
| 可交易性 | 可自由交易 | 不可交易、不可转让 |
| 时代背景 | 物资稀缺时代 | 物资充裕时代 |
| 分配逻辑 | 按支付能力分配资源 | 按贡献度分配话语权 |
| 积累效应 | 可无限积累（贫富分化） | 自然衰减（防止固化） |

## 2.5 讨论备注

> 本章建立的逻辑框架：
> - 从"后货币时代的根本矛盾"出发，推导出 GMC 的必要性
> - 四个子问题（激励、度量、治理、人机协同）构成了 GMC 的设计目标
> - "贡献 → 记录 → 度量 → 话语权"是整个系统的核心流转逻辑
> - MeriToken 与货币的本质区别在于：货币解决分配问题，MeriToken 解决认同和治理问题
