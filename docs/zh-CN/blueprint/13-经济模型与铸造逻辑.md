# 第十三章 经济模型与铸造逻辑

## 13.1 MeriToken 不是货币

再次强调 MeriToken 的经济定位：

- 不可交易、不可兑换
- 没有投机价值
- 不是交换媒介
- 是纯粹的贡献度量和话语权载体

因此，传统货币经济学的约束（通胀控制、货币政策）不适用于 MeriToken。

## 13.2 铸造方案选择

> 讨论中评估了三种方案：

| 方案 | 描述 | 优势 | 劣势 |
|------|------|------|------|
| 总量固定 | 预设上限 | 简单 | 后来者获取难度增大，不公平 |
| 周期限量 | 每周期固定铸造量 | 控制总量 | 贡献变成零和博弈 |
| **无上限 + 衰减自平衡** | 按需铸造，衰减自动销毁 | 公平、无后来者劣势 | 需要衰减模型精确 |

### 选择：无上限铸造 + 衰减自平衡

理由：
- Merit 不是货币，不需要稀缺性维持价值
- 代表"当前活跃贡献度"，衰减保证了这一点
- 避免后来者的不公平劣势
- 话语权基于占比，总量变化不影响治理公平性

## 13.3 为什么不会超发

> 讨论中的关键问题：Merit 凭空产生，会不会超发？

回答：
1. **衰减是天然的销毁机制**：旧 MeriToken 持续衰减
2. **动态平衡**：当铸造速率 = 衰减速率时，总量趋于稳定
3. **占比决定话语权**：即使总量增加，个人话语权取决于占比而非绝对值
4. **类比**：学术引用次数没有上限，但旧论文影响力自然衰减，系统自平衡

## 13.4 动态平衡

### 稳态

用户数稳定时：全网 MeriToken 总量 ≈ 常数

### 增长期

新用户增加 → 总量增长 → 但人均趋于稳定 → 话语权占比自然稀释

### 衰退期

活跃用户减少 → 铸造减少、衰减继续 → 总量下降 → 剩余活跃者占比上升

## 13.6 激励分析

MeriToken 不可交易，但提供的激励是：

| 激励 | 描述 |
|------|------|
| 话语权 | 在社群决策中的影响力 |
| 社会认同 | 高 MeriToken = 高公信力 |
| 优先权 | 某些资源或机会的优先分配 |
| 传承价值 | 可部分传承给后代 |

在后货币时代，社会认同和话语权本身就是最强的激励。

## 13.7 讨论备注

> 经济模型的核心洞察：
> - MeriToken 不是货币，不需要货币经济学的约束
> - 衰减是最优雅的"销毁"机制——无需人为干预，自然平衡
> - 话语权基于占比，使得总量变化不影响治理公平性
> - 这个模型的核心优势：简单、自平衡、公平
> - 不需要复杂的"货币政策"来维护稳定
