# 第十三章 經濟模型與鑄造邏輯

## 13.1 MeriToken 不是貨幣

再次強調 MeriToken 的經濟定位：

- 不可交易、不可兌換
- 沒有投機價值
- 不是交換媒介
- 是純粹的貢獻度量和話語權載體

因此，傳統貨幣經濟學的約束（通膨控制、貨幣政策）不適用於 MeriToken。

## 13.2 鑄造方案選擇

> 討論中評估了三種方案：

| 方案 | 描述 | 優勢 | 劣勢 |
|------|------|------|------|
| 總量固定 | 預設上限 | 簡單 | 後來者取得難度增大，不公平 |
| 週期限量 | 每週期固定鑄造量 | 控制總量 | 貢獻變成零和博弈 |
| **無上限 + 衰減自平衡** | 按需鑄造，衰減自動銷毀 | 公平、無後來者劣勢 | 需要衰減模型精確 |

### 選擇：無上限鑄造 + 衰減自平衡

理由：
- Merit 不是貨幣，不需要稀缺性維持價值
- 代表「當前活躍貢獻度」，衰減保證了這一點
- 避免後來者的不公平劣勢
- 話語權基於佔比，總量變化不影響治理公平性

## 13.3 為什麼不會超發

> 討論中的關鍵問題：Merit 憑空產生，會不會超發？

回答：
1. **衰減是天然的銷毀機制**：舊 MeriToken 持續衰減
2. **動態平衡**：當鑄造速率 = 衰減速率時，總量趨於穩定
3. **佔比決定話語權**：即使總量增加，個人話語權取決於佔比而非絕對值
4. **類比**：學術引用次數沒有上限，但舊論文影響力自然衰減，系統自平衡

## 13.4 動態平衡

### 穩態

使用者數穩定時：全網 MeriToken 總量 ≈ 常數

### 成長期

新使用者增加 → 總量成長 → 但人均趨於穩定 → 話語權佔比自然稀釋

### 衰退期

活躍使用者減少 → 鑄造減少、衰減繼續 → 總量下降 → 剩餘活躍者佔比上升

## 13.6 激勵分析

MeriToken 不可交易，但提供的激勵是：

| 激勵 | 描述 |
|------|------|
| 話語權 | 在社群決策中的影響力 |
| 社會認同 | 高 MeriToken = 高公信力 |
| 優先權 | 某些資源或機會的優先分配 |
| 傳承價值 | 可部分傳承給後代 |

在後貨幣時代，社會認同和話語權本身就是最強的激勵。

## 13.7 討論備註

> 經濟模型的核心洞察：
> - MeriToken 不是貨幣，不需要貨幣經濟學的約束
> - 衰減是最優雅的「銷毀」機制——無需人為干預，自然平衡
> - 話語權基於佔比，使得總量變化不影響治理公平性
> - 這個模型的核心優勢：簡單、自平衡、公平
> - 不需要複雜的「貨幣政策」來維護穩定
