# 第1章：時代判断とビジョン

## 1.1 私たちはどのような時代に向かっているのか

AIは指数関数的なペースで進化しています。予見可能な未来が形を成しつつあります：

- すべての人が自分自身のデジタルアーマー — iFay（パーソナルAIエージェント）を持つようになります
- ほとんどの生産的労働はAIとロボットによって遂行されます
- 生産性は指数関数的に成長し、物質的財は希少から豊富へと転換します

これはSFではありません — すでに進行中のトレンドの自然な延長です。

## 1.2 なぜ通貨は機能しなくなるのか

通貨の本質は物質的財を交換するための媒体です。その存在は一つの前提に依拠しています：**物質的財は希少であり、人々は分配を実現するために交換を必要とする。**

この前提が崩れるとき：

```
希少 → 交換の必要性 → 通貨の必要性
豊富 → 交換の必要性の低下 → 通貨の役割が代替される
```

AIとロボットが生産性を指数関数的に向上させると：
- 基本的な生活必需品の入手はもはや問題ではなくなります
- 物質的交換の必要性は劇的に低下します
- 「生活必需品を得るための資格証明」としての通貨の機能は徐々に意味を失います
- 人々が働き貢献する動機は、もはや金銭的報酬を得ることではなくなります

## 1.3 ポスト通貨時代の核心問題

生存がもはや労働の動機でなくなるとき、一連の根本的な問いが浮上します：

1. **インセンティブ問題**：なぜ人々はなお貢献するのか？
2. **測定問題**：社会に対する個人の価値をどのように認識し記録するのか？
3. **ガバナンス問題**：社会的意思決定における発言権は何を基準に配分すべきか？
4. **人間と機械の問題**：Fayがほとんどの仕事を担うとき、人間とFayの貢献をどのように公平に測定するのか？

これらの問いへの答えこそが、GMCが存在する理由です。

## 1.4 GMCのビジョン

Global Merit Chain（GMC）は、ポスト通貨時代のために設計された社会インフラです。

**核心命題：通貨が機能しなくなったとき、社会的認知と発言権が人間の貢献の主要な原動力となります。GMCはこの目的のために、公正で改ざん不可能な測定基盤を提供します。**

具体的には：

1. **貢献は認知に等しい**：すべての貢献は忠実に記録され認められます。社会的認知が努力への報酬となります
2. **動的な評判**：貢献は時間とともに減衰します。過去の実績に基づいて永続的に発言権を保持することはできません
3. **人間と機械の協調ガバナンス**：人間とFayは同一のシステム内で公平に測定されます
4. **分散型の発言権**：発言権は真の貢献から生まれ、資本、権力、歴史的特権からは生まれません

## 1.5 GMCでないもの

| GMCでないもの | 理由 |
|------------|--------|
| 通貨の代替物 | MeriTokenは取引不可であり、交換媒体ではありません |
| 既存の信用システムの改良 | まったく新しい時代を対象としており、現在へのパッチではありません |
| 投資商品や金融商品 | 投機的価値はなく、売買できません |
| ユートピア的空想 | すでに進行中の技術トレンドに基づいた、実用的な長期戦略です |

## 1.6 未来への基盤づくり

GMCの創設は**未来への基盤づくり**です。

現在のバージョンは議論草案であり、短期的に実装を急ぐものではありません。しかし、今から設計を始める必要があります。なぜなら：
- 社会インフラの構築には長期的な蓄積が必要です
- コンセンサスの形成には時間がかかります
- 技術アーキテクチャの選択は今後数十年の進化の方向性に影響を与えます

私たちは今日の問題を解決しているのではありません — 明日の社会の基盤を構築しているのです。

## 1.7 議論メモ

> 本章で確立された核心的理解：
> - 対象とする時代は現在の社会ではなく、AI普及後のポスト通貨時代です
> - 核心的前提：生産性が大幅に豊富になれば、通貨の役割は代替されます
> - GMCの原動力は「既存の信用システムへのパッチ」ではなく、「ポスト通貨時代のための社会的測定基盤の提供」です
> - 人間の努力の動機は「生存の必要性」から「社会的認知と自己実現」へと移行します
> - これは長期戦略であり、現段階は議論草案です
