# 第2章：核心課題と解決アプローチ

## 2.1 問題の全体像

ポスト通貨時代が直面する核心的問題は、一つの根本的矛盾に集約できます：

**生存がもはや労働の動機でなくなったとき、社会はどのように貢献にインセンティブを与え、貢献を記録し、貢献に基づいてガバナンスを行うのか？**

この根本的矛盾は4つのサブ問題に分解されます：

### 問題1：インセンティブ

金銭的報酬がなければ、なぜ人々はなお貢献するのか？

回答：社会的認知です。人間は本質的に承認、尊敬、影響力を渇望します。GMCは貢献を可視的で定量化可能な社会的認知（MeriToken）に変換し、それに実質的な発言権を付与します。

### 問題2：測定

貢献をどのように公平に記録し測定できるのか？

回答：分散型コンセンサスに基づく記録です。ブロックチェーンと暗号技術を活用し、偽造不可能で改ざん不可能で否認不可能な貢献記録ネットワークを構築します。いかなる単一機関も記録を歪めることはできません。

### 問題3：ガバナンス

発言権は何を基準に配分すべきか？

回答：貢献に基づく動的配分です。発言権はコミュニティ内でのMeriTokenのシェアから導出され、MeriTokenは減衰します — つまり発言権は継続的な貢献によって維持されなければならず、永続的な特権は存在しません。

### 問題4：人間と機械の協調

Fayがほとんどの仕事を担うとき、人間とFayの貢献をどのように公平に測定するのか？

回答：統一的な測定システムです。人間とFayは同一のシステム内で記録・測定されます。Fayの貢献はその人間原型または所属組織に帰属します。

## 2.2 解決アプローチ

GMCのアプローチは以下のように要約できます：

```
貢献 → 記録 → 測定 → 発言権
```

1. **貢献が発生する**：人間またはFayが社会に価値ある行動を行います
2. **貢献が記録される**：コンセンサスメカニズムを通じて、貢献はオンチェーンに不変的に記録されます
3. **貢献が測定される**：貢献はMeriTokenに変換され、減衰とフロア値を持ちます
4. **発言権が発効する**：MeriTokenのシェアがコミュニティガバナンスにおける影響力を決定します

## 2.3 6つの設計原則

| 原則 | 意味 | 対処する問題 |
|-----------|---------|-------------------|
| 偽造不可能 | 貢献記録はコンセンサス検証を通過しなければならない | 不正な貢献の防止 |
| 改ざん不可能 | 履歴記録は永久に保存され変更できない | 記録の信頼性の確保 |
| 否認不可能 | 貢献者は自身の行動を否認できない | 責任追跡の実現 |
| 分散型 | いかなる単一機関にも依存しない | 権力独占の防止 |
| 動的測定 | 貢献は時間とともに減衰する | 継続的参加のインセンティブ |
| 普遍的参加 | すべての人間とFayに開放 | 公平性 |

## 2.4 通貨システムとの根本的な違い

| 次元 | 通貨 | MeriToken |
|-----------|----------|-----------|
| 本質 | 交換媒体 | 貢献測定 |
| 原動力 | 生存の必要性 | 社会的認知と自己実現 |
| 取引可能性 | 自由に取引可能 | 取引不可、譲渡不可 |
| 時代背景 | 物質的希少の時代 | 物質的豊富の時代 |
| 分配ロジック | 購買力によって資源を配分 | 貢献によって発言権を配分 |
| 蓄積効果 | 無限に蓄積可能（富の二極化） | 自然に減衰（固定化の防止） |

## 2.5 議論メモ

> 本章で確立された論理的フレームワーク：
> - 「ポスト通貨時代の根本的矛盾」から出発し、GMCの必要性を導出しています
> - 4つのサブ問題（インセンティブ、測定、ガバナンス、人間と機械の協調）がGMCの設計目標を構成します
> - 「貢献 → 記録 → 測定 → 発言権」がシステム全体の核心的フローロジックです
> - MeriTokenと通貨の根本的な違い：通貨は分配問題を解決し、MeriTokenは認知とガバナンスの問題を解決します
